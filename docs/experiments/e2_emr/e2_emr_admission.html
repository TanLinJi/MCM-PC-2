2. 我们现在做了什么？

原始 Point-Cache 在 positive cache 中更偏向保存低熵样本。E2-EMR 的改动很小：不改 backbone、不训练模型、不改 final logits 融合，只改变 positive global/local cache 的样本排序与替换规则。

Baseline: lower entropy → better cache entry E2-EMR: higher reliability score → better cache entry

Reliability v1 文件中已经显式加入 compute_prediction_margin() 和 compute_reliability_score()，并在在线测试循环中用 reliability score 更新 positive cache。

3. top-1/top-2 margin 是什么？

模型会对每个类别输出概率。top-1 是概率最高的类别，top-2 是概率第二高的类别。它们的差值就是 margin：

M(x) = p(1)(x) − p(2)(x)

如果 margin 大，说明模型明显更相信第一类；如果 margin 小，说明第一类和第二类很接近，样本处于混淆边界附近。

4. Reliability Score 公式

R(x) = −H(x) + λmM(x)
符号含义作用
R(x)样本 x 的可靠性分数越大越适合进入 positive cache
H(x)预测熵 / uncertainty越大越不确定，因此前面加负号
M(x)top-1/top-2 probability margin越大表示类别区分越清楚
λmmargin 项权重当前 v1 使用 1.0
直观解释：一个可靠的 cache 样本应该同时满足“预测熵低”和“top-1 明显高于 top-2”。

8. 分析结论

正向信号：entropy + margin 在 add_global_2 和 dropout_global_2 上提升，说明 margin 对全局类别混淆有帮助。

主要问题：它在 add_local_2、dropout_local_2、rotate_2、scale_2 上下降，说明 class-level margin 无法可靠处理局部结构损坏和几何变换。

因此，E2-EMR 的定位不是最终方法，而是一个关键 ablation：它证明“只加入分类 margin 仍不够”，从而自然引出下一阶段的 global-local consistency。

9. 可写进论文的英文素材

[Section: Experiments / Ablation Study]

A simple entropy-margin reliability score improves global corruption cases but degrades under local and geometric corruptions. This indicates that class-level confidence alone is insufficient for robust 3D test-time cache admission. Reliable point-cloud cache construction should further consider the structural consistency between global object features and local part evidence.

[Section: Method Motivation]

Although top-1/top-2 margin helps identify less ambiguous predictions, it remains a class-level criterion and ignores whether the global object prediction is supported by local part evidence. This motivates our next reliability signal: global-local structural consistency.
