MCM-PC · E4-CANC-v0 规则解释与阶段分析

主题：为什么当前 conservative negative cache 是安全但保守的，以及下一步为什么要进入 E4-CANC-v1。

Stage: E4 Experiment: E4-CANC-v0 Rule: Conservative Negative Cache Dataset: ModelNet-C Result: +0.02 Avg.

1. 当前实验在做什么

E4-CANC-v0 的目标不是继续改 positive cache，而是利用 E4-CANC-DIAG-v2 发现的规律： global-local conflict 可以显著提高 global pseudo-label 出错概率。因此，E4-CANC-v0 尝试把这类冲突样本用于 negative cache，也就是抑制可能错误的 global predicted class。

核心原则： local alternative class \(k_l^\star\) 不作为 pseudo-label。E4 只使用冲突信号来判断 \(\hat y_g\) 是否应该被 suppress。

当前规则为：

\[ I_{neg}^{E4}(x)=I_H(x)\lor(I_M(x)\land I_D(x)). \]

2. 三个判断条件分别是什么意思

2.1 \(I_H\)：原始 Point-Cache 的 entropy negative 条件

\[ I_H(x)=\mathbf{1}[\tau_l < H_g(x)<\tau_u]. \]

\(I_H\) 表示原始 Point-Cache 已经会选进 negative cache 的样本。它对应的是中等 entropy 区间：模型不是非常确定，也不是完全混乱，可能处在不稳定区域。

2.2 \(I_M\)：global boundary candidate

\[ I_M(x)= \mathbf{1}[M_g(x)<\tau_m]\cdot \mathbf{1}[p_g^{(1)}(x)>\tau_p]. \]

其中：

\[ M_g(x)=p_g^{(1)}(x)-p_g^{(2)}(x). \]

\(I_M\) 表示 global top-1 和 top-2 比较接近，同时 top-1 概率又不是太低。因此它不是完全混乱样本，而是可能位于类别边界附近的样本。

2.3 \(I_D\)：global-local conflict

\[ I_D(x)=\mathbf{1}[D_{gl}^{mean}(x)>\tau_d]. \]

其中 mean-local conflict 为：

\[ D_{gl}^{mean}(x)= \left[ \max_{k\ne \hat y_g}p_l^{mean}(k\mid x) - p_l^{mean}(\hat y_g\mid x) \right]_+. \]

\(I_D\) 表示 local part evidence 明显不支持 global predicted class \(\hat y_g\)。

3. E4-CANC-v0 规则的集合含义

当前 E4-CANC-v0 使用：

\[ I_{neg}^{E4} = I_H \lor (I_M\land I_D). \]

这句话可以拆成两部分：

部分含义
\(I_H\) 保留原始 Point-Cache 的 negative cache 样本。
\(I_M\land I_D\) 额外加入“既是 global 边界样本、又存在 global-local 冲突”的样本。
Universe: all test samples I_H: original entropy-based negative samples I_M: global boundary samples I_D: global-local conflict samples E4-CANC-v0 selects: I_H ∪ (I_M ∩ I_D) New samples compared with baseline: (not I_H) ∩ I_M ∩ I_D

也就是说，E4-CANC-v0 并不是把所有 conflict 样本都加入 negative cache，而只加入其中非常保守的一部分。

4. 为什么真正新增部分是 \((\neg I_H)\land I_M\land I_D\)

baseline negative cache 已经包含：

\[ I_{neg}^{base}=I_H. \]

E4-CANC-v0 包含：

\[ I_{neg}^{E4}=I_H\lor(I_M\land I_D). \]

因此，E4 相比 baseline 真正新增的样本是：

\[ I_{new} = I_{neg}^{E4}\land \neg I_{neg}^{base}. \]

代入定义：

\[ I_{new} = \bigl(I_H\lor(I_M\land I_D)\bigr)\land \neg I_H. \]

利用分配律：

\[ I_{new} = (I_H\land\neg I_H) \lor (I_M\land I_D\land\neg I_H). \]

因为：

\[ I_H\land\neg I_H=0, \]

所以：

\[ I_{new} = \neg I_H\land I_M\land I_D. \]
结论： E4-CANC-v0 真正新增的样本，必须是 baseline 本来不会选入 negative cache，但同时满足 boundary 和 conflict 的样本。

6. 为什么说 v0 安全但保守

6.1 安全性

E4-CANC-v0 不使用所有 conflict 样本，而只使用 \(I_M\land I_D\)。这降低了 false negative cache pollution 的风险。

\[ I_M\land I_D \Rightarrow \text{global boundary} + \text{local disagreement} \]

这比单独使用 \(I_D\) 更谨慎。

6.2 保守性

但是这个规则太严格，导致真正新增样本比例只有约 \(0.78\%\)。

\[ I_{new}=\neg I_H\land I_M\land I_D. \]

因此，即使 conflict signal 本身对错误 global pseudo-label 有很强诊断能力，它对最终 cache 的影响仍然很有限。

7. 下一步为什么要做 E4-CANC-v1

E4-CANC-v0 的结果说明 negative suppression 路线比 E2/E3 的 positive admission 路线更稳定，但当前规则过于保守。下一步需要放宽规则，让更多经过诊断验证的 conflict 样本参与 negative suppression。

建议 E4-CANC-v1 规则为：

\[ I_{neg}^{E4-v1}(x) = I_H(x) \lor \left( I_D(x) \land p_g^{(1)}(x)>\tau_p \right). \]

也就是去掉 \(I_M\) 的小 margin 限制，只保留：

  • global-local conflict；
  • global top-1 不是完全低置信；
  • 仍然 suppress global predicted class \(\hat y_g\)，不使用 local alternative class 作为 pseudo-label。
预期： E4-CANC-v1 会比 v0 使用更多 conflict samples，从而增强 negative suppression 的实际作用。

8. 论文中应如何表述

[Method Motivation]

The conservative CANC rule preserves the original entropy-based negative cache and only adds samples that are both globally ambiguous and locally conflicting. This design avoids aggressive false-negative insertion but may under-utilize the diagnostic power of global-local conflict.

[Ablation Analysis]

E4-CANC-v0 slightly improves the full-corruption average, indicating that conflict-aware negative suppression is safer than positive cache promotion. However, the newly added negative samples are less than 1% on average, showing that the rule is under-active.

[Next Variant]

To strengthen the effect of conflict-aware suppression, E4-CANC-v1 relaxes the boundary constraint and admits samples with reliable global-local conflict and non-trivial global confidence.
